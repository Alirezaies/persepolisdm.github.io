What Is Persepolis?

Persepolis Is A GUI For Aria2 Which Is Powered By Python.
Persepolis Is A Sample Of Free And Open Source Software And Developed For GNU/Linux Distributions.
If You Also Using Any GNU/Linux Distribution You Can Join Persepolis Users And Help Us With Developing It.

Installation

Ubuntu and other Debian distro

Here We Described How To Install Persepolis On Debian And Debian Based Distros

You can also simply run these commands :

sudo add-apt-repository ppa:persepolis/ppa
sudo apt update
sudo apt install persepolis

Arch and other Arch bases

for installing latest release (recommand) :

yaourt -S persepolis

or for latest change on github :

yaourt -S persepolis-git
Obviously you can't install both of persepolis and persepolis-git simoltaneously . and you must uninstall one of them(with pacman) for installing another one.

Firefox Integration

First,Install Flashgot From This Link
Then In Flashgot Setting Open Prefrences And Add The "Persepolis Download Manager"
In The Executable Path Type This:

/usr/bin/persepolis

In The Command Line Argument Template,Write This:

[--link URL][--name FNAME][--referer REFERER][--headers HEADERS][--agent UA][--cookie COOKIE]


In The Advanced Tab,Disable Automatic Download Manager Detection
Also In Flashgot Media Tab Select Persepolis Download Manager
