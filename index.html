What Is Persepolis?

Persepolis Is A Download Manager And A GUI For Aria2 Which Is Powered By Python.
Persepolis Is A Sample Of Free And Open Source Software And Developed For GNU/Linux Distributions And Mac OS.
If You Also Using Any GNU/Linux Distribution And Mac OS You Can Join Persepolis Users And Help Us With Developing It.

Installation

Ubuntu and other Debian distro

Here We Described How To Install Persepolis On Debian And Debian Based Distros

You can also simply run these commands :

sudo add-apt-repository ppa:persepolis/ppa
sudo apt update
sudo apt install persepolis

Arch and other Arch bases

for installing latest release (recommand) :

yaourt -S persepolis

or for latest change on github :

yaourt -S persepolis-git
Obviously you can't install both of persepolis and persepolis-git simoltaneously . and you must uninstall one of them(with pacman) for installing another one.

Fedora

sudo dnf copr enable amirsamimi/persepolis
sudo dnf install persepolis

Mac OS X

you must install aria2 python3 python3-requests python3-pyqt5 first! you can use Brew .

brew install python3
brew install aria2
brew install pyqt5
pip3 install requests

an alternative way for installing dependencies in mac , after cloning project you can run dependay_mac in terminal

when dependencies installed :

git clone "https://github.com/persepolisdm/persepolis.git"
cd persepolis
bash install mac

you can find Persepolis Download Manager Icon in your Application folder and applications menu .

Firefox Integration

First,Install Flashgot From This Link
Then In Flashgot Setting Open Prefrences And Add The "Persepolis Download Manager"
In The Executable Path For GNU/LINUX Type This:

/usr/bin/persepolis

And For Mac OS X Type :

/usr/local/bin/persepolis

In The Command Line Argument Template,Write This:

[--link URL][--name FNAME][--referer REFERER][--headers HEADERS][--agent UA][--cookie COOKIE]


In The Advanced Tab,Disable Automatic Download Manager Detection
Also In Flashgot Media Tab Select Persepolis Download Manager


Disable Autostart Downloads
